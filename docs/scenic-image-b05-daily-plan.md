## 二、单条素材验收标准（验收人打勾）

每完成一条，验收人按下列 8 项逐项核对：

- [ ] 图地一致：图片确为该景点，不混用其他地点
- [ ] 时效合理：非明显过期/季节错位素材（除特殊景观）
- [ ] 清晰可用：主视觉清晰，非严重压缩或水印遮挡
- [ ] 权限合规：来源可追溯，授权/许可可留档
- [ ] 命名正确：资源名为 `scenic_<id>`，与数据 ID 一致
- [ ] 数据同步：`scenics.json` 对应条目改为 `imageMatchType: exact`
- [ ] UI复核：卡片与详情页均显示“实拍已核验”
- [ ] 门禁复检：执行 `python3 scripts/verify_scenic_images.py --gate-release`
