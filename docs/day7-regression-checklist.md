# Day7 回归验收清单

## 一、今日任务卡

- [ ] 首页显示「今日任务」卡片，含 4 项任务与进度条
- [ ] 小图表显示 `词/闯/练/复` 四个柱形与总完成率百分比
- [ ] 完成 4 项后出现「领取今日奖励 +120 XP」
- [ ] 奖励领取后显示「今日奖励已领取」，当天不可重复领取

## 二、错题复习调度

- [ ] 闯关答错后，错题写入「易混淆词」
- [ ] 同时创建复习计划（1/3/7 天节奏）
- [ ] 首页待复习数量与实际到期条目一致

## 三、今日待复习页

- [ ] 首页「去复习」进入「今日待复习」
- [ ] 页头可切换「连刷模式 / 列表模式」
- [ ] 连刷模式下可使用「完成并下一条」
- [ ] 列表模式下每条可点「完成」
- [ ] 完成后出现成功提示，并自动推进下一轮或结束

## 四、易混淆词详情

- [ ] 到期条目标记「待复习」
- [ ] 详情页可点「完成本次复习」
- [ ] 点击后任务「复习一次」计数 +1

## 五、稳定性

- [ ] 多次进入/退出页面后进度不丢失
- [ ] 当天日期内刷新 App 后状态保持一致
- [ ] 第二天自动切换到新的任务桶（旧日数据不覆盖）
