# 唠嗑小馆 Android SDK 隐私台账（提审用）

## 使用说明

将下表补齐后同步到隐私政策页面，并在各商店后台填写一致信息。

## SDK 清单

| SDK 名称 | 供应商 | 用途 | 收集信息 | 是否必要 | 隐私政策链接 |
| - | - | - | - | - | - |
| 高德地图 Android SDK | 高德软件有限公司 | 地图展示、定位与导航 | 设备标识、位置信息、网络状态 | 是（打卡功能） | 待补充 |
| 讯飞语音评测 SDK | 科大讯飞股份有限公司 | 发音评测、语音识别 | 麦克风音频、设备信息 | 是（口语练习） | 待补充 |
| Firebase Crashlytics / Bugly（二选一） | Google / 腾讯 | 崩溃分析 | 设备型号、系统版本、崩溃堆栈 | 否（建议） | 待补充 |

## 权限台账

| 权限 | 场景 | 拒绝后影响 | 触发时机 |
| - | - | - | - |
| `RECORD_AUDIO` | 口语练习发音评测 | 无法录音评测 | 用户进入口语练习并点击录音 |
| `ACCESS_FINE_LOCATION` | 风景打卡距离判定 | 无法定位打卡 | 用户进入打卡页并点击开始打卡 |
| `CAMERA` | 拍照打卡（可选） | 无法现场拍照，仅可上传本地图片 | 用户点击拍照按钮 |

## 合规检查点

- 权限必须延迟到业务触发时申请，禁止首次启动一次性申请。
- 拒绝权限后需提供可理解的降级路径和再次授权入口。
- 隐私政策中的 SDK/权限/用途说明必须与 App 内行为一致。
