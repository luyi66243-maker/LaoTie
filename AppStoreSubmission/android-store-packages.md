# 五大商店提审材料包说明

建议按如下目录准备并上传：

```text
AppStoreSubmission/
  AndroidPackages/
    Huawei/
    Xiaomi/
    OPPO/
    vivo/
    Tencent/
```

每个目录建议包含：

1. `app-release.apk` 或 `app-release.aab`
2. `metadata.txt`（应用简介、更新说明、关键词）
3. `privacy_policy_url.txt`
4. `software_copyright.pdf`
5. `app_filing_record.pdf`
6. `screenshots/`（5 张 1080x1920）

## Google Play（第二批）

`GooglePlay/` 目录建议额外准备：

- Data Safety 表单草稿
- 内容分级说明
- 政策合规自检记录
